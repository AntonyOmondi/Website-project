# Tony's Coffee Photography

Welcome to **Tony's Coffee Photography**, a website dedicated to celebrating the beauty of coffee culture through photography. This project combines HTML and CSS to create a visually appealing and user-friendly web experience.

## Features

- **Header Navigation**: Easily access sections like Home, Types, About Us, Reviews, and Contact Us.
- **Main Content**: A detailed introduction to coffee photography and its essence.
- **Coffee Types**: A showcase of various coffee styles with high-quality images.
- **About Us Section**: Information about the creators and the mission behind the website.
- **Reviews Section**: User testimonials showcasing experiences and feedback.
- **User Review Form**: A form for visitors to share their thoughts.
- **Contact Details**: Social media links, contact information, and an embedded map for location.

## Folder Structure

```
project/
├── index.html       # Main HTML file
├── style.css        # CSS styling file
├── images/          # Directory for images used in the project
```

## Technologies Used

- **HTML5**: Structuring the content of the website.
- **CSS3**: Styling the website with a focus on layout, typography, and responsiveness.
- **Google Fonts**: Roboto font family for a modern look and feel.

## Setup Instructions

1. Clone this repository or download the files.
2. Place the files in a directory on your local machine.
3. Open `index.html` in your favorite browser to view the website.

## Key CSS Highlights

- **Typography**: Roboto font ensures a clean and professional appearance.
- **Flexbox Layouts**: Used extensively for responsive and flexible design.
- **Hover Effects**: Interactive styling for images, buttons, and navigation links.
- **Custom Colors and Shadows**: A warm coffee-inspired theme with rich shadows for depth.

## Credits

- **Images**: Images are stored in the `images/` directory and are integral to the website's visual appeal.
- **Map**: Google Maps iframe embedded for the contact section.
